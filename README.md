### Prerequisites

* [Flutter SDK](https://flutter.dev)
* [VS Code IDE](https://code.visualstudio.com/) recommended

### Installation

1. Clone the repo
```
git clone https://github.com/nemoryoliver/znn_playground.git
```
2. Install package dependencies
```
flutter pub get
```
3. Run
```
flutter run
```
